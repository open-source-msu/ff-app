# Приложение Твой ФФ!

Приложение физического факультета МГУ, предоставляющее доступ к расписанию и другим
сервисам для студентов. Приложение разрабатывается студентами физфака МГУ при поддержке
профкома студентов физического факультета МГУ.

Основная организация: https://github.com/profcomff
